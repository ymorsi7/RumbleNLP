About This Project

This is my first NLP project, where I try to detect hate speech on videos relating to sexism on the right-wing platform, Rumble.

  • NLP can be used to automatically analyze and structure text (quickly and cost-effectively).
  • Using TF-IDF vectorization, we located the most significant words in each comment section, and overall.
  • Subsequently, we trained a model to classify hate speech using data extracted from the site using a decision tree.
  • Because Rumble specifies that "systematic" retrieval of data is prohibited, I needed to scrape the data manually.
  • As someone who loves automation, this was a tough task, but it needed to be done.
  • To find sexist content on Rumble, I selected five videos relating to sexism on the platform:

    TF-IDF Vectorization

    The TF-IDF is calculated by multiplying the term frequency and inverse document frequency.

    Decision Tree Model

    After using scikit-learn's DecisionTreeClassifier() function, I trained the model on the data, and then tested it on the test data. The model achieved an accuracy of about 0.89.

    Word Clouds

    Using the Google Sheets extensions MeaningCloud and ChartExpo, I was able to create word clouds from extracted topics, deep categorization, text clustering, and IPTC text classification.
